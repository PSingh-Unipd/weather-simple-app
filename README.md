# weather-simple-app

### About this repo
Simple weather application build using Angular v13. All unit tests are done using Jasmine/Karma with 100% of code coverage.

### Installation
```
npm install
```
### Browse
```
ng serve
```
### Build Prod
```
npm run build --prod
```
